Demo / 开发阶段的调试入口（清档、debug 开关、关卡跳转等）要做成**肉眼可见的按钮**，禁止用长按、多指点击、快捷键等隐藏手势。

**Why**：demo 还没上线，不需要防误触；隐藏手势反而让自己测试时也费劲。曾经在 ProjectDrop 做了"长按标题 3 秒清档"，用户反馈"太傻了"，改成了可见按钮后效率高很多。

**How to apply**：dev 调试工具默认放在主菜单或游戏内明显位置，按钮文字注明 `（dev）`。等正式构建再用 `GameConfig.DEBUG_MODE` 开关一次性屏蔽。不要默认就走隐藏化设计。
