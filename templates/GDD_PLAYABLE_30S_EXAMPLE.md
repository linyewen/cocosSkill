## Ch 1 · 核心体验

### 1.1 一句话玩法
> ✏️ 玩家**左右滑动**控制战机，**自动射击**击碎下落方块，**打道具升级**变强，**击败 Boss** 获胜。

### 1.2 情绪曲线
> ✏️
```
开头（0-5s）：新手紧张，学习操作，小怪爽送
中段（5-24s）：快速成长，火力爆炸，爽感累积
尾段（24-30s）：Boss 决战，紧张但可胜
```

### 1.3 核心爽点
> ✏️
```
1. 满级多炮（4 发）+ 最强攻击 + 最快射速，一瞬间打碎一整排高 HP 方块墙
2. 暴击 3 倍伤害时跳出大红字"CRITICAL!"的反馈
3. 击杀 Boss 的全屏爆炸 + 震屏 + 胜利界面
```

## Ch 2 · 玩家操作

> ✏️ 手指在屏幕下半部左右滑动控制战机 X 轴位置（战机只有水平移动）；战机自动向上射击不需手动点；金币自动吸附玩家（靠近 100px 内自动飞向战机）。

## Ch 3 · 游戏单位

### 单位 1：战机
> ✏️
```
职责：玩家操控的主角，向上射击消灭敌人
生成规则：游戏开始时场景常驻
行为描述：跟随玩家手指水平移动；自动向上发射子弹
视觉特征：3 档外观对应攻击等级 Lv1/Lv2/Lv3，外观越高级越威武
美术资源：见 assets/player/1.png / 2.png / 3.png
销毁条件：不销毁（失败时停止动作即可）
```

### 单位 2：子弹
> ✏️
```
职责：战机发射的攻击弹药
生成规则：战机自动发射（按射速间隔）
行为描述：直线向上飞行，命中方块造成伤害
视觉特征：3 档外观（对应玩家等级），越高级越大越亮
美术资源：见 assets/bullet/bullet_*.png
销毁条件：命中方块 或 飞出屏幕上方
```

### 单位 3：普通方块（敌人）
> ✏️
```
职责：从上方下落的敌人，玩家主要的射击目标
生成规则：按波次从屏幕顶部生成，一波 5-15 个不等
行为描述：以 "step-down" 方式下落 —— 停顿一段 → 瞬移下降一格 → 循环
         HP 不同用不同颜色区分（8 档）
视觉特征：8 档颜色梯度，从绿 → 黄 → 红代表 HP 从低到高
美术资源：见 assets/block/c1.png ~ c8.png
销毁条件：HP 归 0（被子弹打死）→ 触发金币掉落
失败触发：方块落到屏幕底部 = 游戏失败
```

### 单位 4：道具方块
> ✏️
```
职责：特殊的方块，打破后让玩家升级
生成规则：混在普通方块波次里（每波夹 1-2 个）
行为描述：HP 较低（8-12），打破后掉出道具图标飞向战机
视觉特征：外观和普通方块不同（带道具图标），3 种不同颜色代表 3 种升级（橙 = 多炮 / 红 = 攻击 / 绿 = 射速）
美术资源：见 assets/powerup/extra_gun.png / attack_up.png / speed_up.png
销毁条件：HP 归 0 → 图标飞向战机触发升级
```

### 单位 5：Boss
> ✏️
```
职责：最终 BOSS，击杀代表胜利
生成规则：最后一波（约 24-25 秒）出现，仅 1 个
行为描述：横跨整排屏幕宽度，HP 很高（需要满级火力才能 5 秒内打死）
         下落速度比普通方块慢
视觉特征：比普通方块大 5 倍，单独一个贴图
美术资源：见 assets/block/boss.png
销毁条件：HP 归 0 = 胜利结算
```

### 单位 6：金币
> ✏️
```
职责：玩家的资源，攒够可升级
生成规则：普通方块死亡时从方块位置抛出 1-3 个
行为描述：带抛物线飞行 0.6 秒落地，落地后停留 3 秒自动消失
         战机靠近 100px 内自动吸附到战机
视觉特征：黄色金币帧动画（6 帧旋转）
美术资源：见 assets/coin/coin_1.png ~ coin_6.png
销毁条件：被战机收集 或 落地 3 秒后自动消失
```

## Ch 4 · 升级成长系统

> ✏️
```
升级路线数：3 条
每条意图：
- 多炮（EXTRA_GUN）：增加炮口数量（1 → 2 → 3 → 4 发子弹），帮玩家解决"方块多扫不完"的问题
- 攻击（ATTACK_UP）：每发子弹伤害提高，帮玩家解决"方块 HP 太高"的问题
- 射速（SPEED_UP）：射击间隔缩短，帮玩家配合多炮实现指数级火力

升级触发：打破混在波次里的道具方块（每种道具方块对应一条升级）
额外触发：攒够金币阈值（一局内有 2 档金币阈值额外升级 1 级）
最大等级：每条路线 3 级

限时：一局内满级就够了，不跨局持久化

关键约束：
- 前 15 秒内必须让玩家攒满 3 种道具路线（否则 18 秒后的爽打期没有爽感）
- Wave 1 / 2 / 3 必须各有 1 个道具方块能拿到
- 道具方块 HP 要低（8-12），新手也能打碎
```

## Ch 5 · 关卡节奏阶段

> ✏️
```
阶段数：5 个

阶段 1：开场教学（约 0-5s）
  - 情绪目标：紧张但友好，让玩家 2 秒内知道怎么操作
  - 主要内容：第一波小方块 + 第一个道具方块（多炮）
  - 教学元素：屏幕上显示一次"左右滑动"手势引导动画，玩家动了就消失

阶段 2：成长期（约 5-12s）
  - 情绪目标：给玩家持续的"变强"反馈
  - 主要内容：2 波中等 HP 方块，每波夹 1-2 个道具方块
  - 玩家预期：收集到多炮 Lv2 / 攻击 Lv1 / 射速 Lv1

阶段 3：满级期（约 12-18s）
  - 情绪目标："我变最强了" 的成就感
  - 主要内容：1 波满级方块配置，让玩家拿到最后 1-2 个升级
  - 玩家预期：满级 3 + 3 + 3 所有升级

阶段 4：爽打期（约 18-24s）
  - 情绪目标：满级火力秀，纯输出的爽感
  - 主要内容：1 波超高 HP 方块墙，但没有道具方块（无干扰纯爽打）

阶段 5：Boss 决战（约 24-30s）
  - 情绪目标：紧张 + 最终胜利
  - 主要内容：Boss 出现，用满级火力 5-6 秒打死

Boss：有，阶段 5 登场
```

**⚠️ 具体每波多少方块、HP 多少、间隔多少秒 —— AI 会自动设计，策划不写**。

## Ch 6 · 反馈系统

> ✏️

| 玩家行为 / 事件 | 期望感受 | 反馈强度 |
|---|---|---|
| 子弹命中方块 | 有打击感 | 方块瞬间放大 1.1 倍抖动 + 扣血数字飘出 + 微震屏（强度：低）|
| 暴击（随机）| 刺激爽感 | 红色大字 "CRIT!" + 金色光圈 + 加强音效（强度：中）|
| 打破道具方块 | 获得感 | 道具图标以抛物线飞向战机 + 收集时闪光 + 清脆音 + 战机外观切换（强度：中-高）|
| 击杀普通方块 | 持续回馈 | 金币抛物线喷出 + 扣血数字 |
| 击杀 Boss | 高潮时刻 | 全屏爆炸粒子 + 强震屏 0.4s + 胜利音 + Boss 爆炸分解动画（强度：最高）|
| 方块落地（失败）| 挫败感 | 屏幕短暂泛红 + 失败音 |
| 升级 | 变强感 | 战机外观切换（攻击等级）+ 子弹外观切换 + 升级光效 |
| 金币收集 | 微反馈 | 数字跳增 + 小"叮"音 |
