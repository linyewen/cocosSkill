# Cocos Creator Claude Skills

AI × Cocos Creator 游戏开发的 Claude Code Skills 和 Commands 集合。

## 目录结构

```
├── CLAUDE.md                    # 全局开发规范
├── skills/                      # Claude Skills（全局）
│   ├── feature-design/SKILL.md  # 功能设计分析
│   └── prefab-analyze/SKILL.md  # Prefab 蓝图推导
└── commands/                    # Claude Commands（项目级）
    ├── cocos-layer.md           # 游戏分层设计
    ├── cocos-migrate.md         # Cocos 2.x → 3.8 迁移检查
    ├── cocos-prefab.md          # 创建预制体
    └── cocos-review.md          # 代码审查
```

## 安装方式

### Skills（全局生效）
复制 `skills/` 下的文件夹到 `~/.claude/skills/`

### Commands（项目级）
复制 `commands/` 下的文件到项目的 `.claude/commands/`

### CLAUDE.md（全局规范）
复制 `CLAUDE.md` 到 `~/.claude/CLAUDE.md`
